## 项目搭建：ToolLabs 在线工具站（已上线MVP）
- 项目名：ToolLabs（toollabs.vercel.app）
- 策略：纯前端工具站（零服务器成本），Vercel 免费部署，Google AdSense 变现
- 第一个工具：PDF to Word 转换器
  - 技术栈：pdf.js 3.11.174 + docx 9.1.0，全部本地化（lib/ 文件夹）
  - 修复了 CDN 地址错误（docx 不在 cdnjs 上，改用 jsDelivr 并下载到本地）
  - 修复了浏览器兼容性问题（改用动态创建 file input 方式，避免隐藏 input 事件不触发）
  - 支持拖拽上传和点击上传
  - 最大支持 100MB
  - 纯前端处理，文件不上传服务器
- 已创建文件：index.html, pdf-to-word.html, css/style.css, js/pdf-to-word.js, vercel.json, lib/pdf.min.js, lib/pdf.worker.min.js, lib/docx.umd.min.js
- 部署方式：需用户自行通过 Vercel 导入部署（3步一键操作）
- 调试过程：解决了 CDN 404 问题、alert() 在预览面板被禁用的问题、file input change 事件不触发的兼容性问题
- 最终版经用户实测：转换 2018 年法考真题 PDF（735KB）成功，生成 DOCX 可正常下载打开 ✅
- 后续扩展计划：压缩图片、JSON格式化、计算器、Merge PDF 等
